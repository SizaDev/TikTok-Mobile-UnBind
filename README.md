# TikTok Phone Unbind CLI

A command-line tool to unbind the phone number linked to a TikTok account using the session ID.

---

## Features

- Unbind phone number linked to TikTok accounts.
- Save session IDs and results locally in `sessions.json`.
- Simple and clean command-line interface.

---

## Requirements

- Python 3.8+
- requests library

Install the library:

pip install requests

---

## Usage

Run the script:

python tiktok_unbind_cli.py

Enter your TikTok session ID when prompted:

Enter TikTok SessionID (or 'exit' to quit):

- Type your session ID to unbind the phone number.
- Type `exit` to quit the program.

Results are displayed in the terminal and saved in `sessions.json`.
